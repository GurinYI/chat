## Установка и Запуск

У вас должнен быть установлен NodeJS v18.18.2


Установка зависимостей

   ```shell
   npm install
   ```

## Запуск по адресу http://localhost:3000/ 

1. Запустить скирпт

```shell
npm run build
npm run start
```
